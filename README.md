# 🚀 Playwright JavaScript Automation Framework

This repository contains a complete end-to-end **UI Automation Testing Framework** built using **Playwright with JavaScript**.  
It follows real-world automation practices using the **Page Object Model (POM)** design pattern.

---

## 📌 Project Overview

This project demonstrates automation of web applications using Playwright including:

- Login / Logout Automation
- Page Object Model (POM)
- Locators (CSS, XPath, Text, Role-based)
- Assertions & Validations
- Dropdown handling
- Alerts handling
- Frames handling
- Window handling
- Auto suggestions handling
- Keyboard & Mouse actions
- File upload & download
- Data-driven testing
- Test execution reports
